成果 ① · q→σ 標定 + WLS/EIV fitter · 理論

回歸自變數含噪 → 誤差在變數(EIV),前向 OLS 有衰減偏差

DAC = k · disparity + b
  • DAC 是精確值(VCM 命令),disparity 是量測值(含噪)——自變數含噪即 EIV。
  • 前向 OLS(對 disparity 回歸)有衰減偏差(把斜率往 0 拉),隨 σ² 放大。
  • 反向 OLS(對 DAC 回歸)消偏,但變異(CV)較大。
q→σ 標定:σ(q) = σ₀ · q−p,橋接權重指數 γ = 2p。 n=10 擬合殘差需 studentized 槓桿修正 |e|/√(1−h_ff)。
fitter特性
ols_forwardv1 預設;衰減偏差
ols_inverse消偏;變異較大
wls_inverse加權(w=qγ);消變異
demingδ=0 ≡ inverse、δ→∞ ≡ forward
Deming(δ=0)與反向逐列位元級一致(UT 釘死)——主表 4 個 σ 下 deming 與 ols_inverse 完全相同,驗證通過。

成果 ① · 結論與建議

維持 forward 預設;σ≥1 才條件式啟用 WLS(γ=1)

建議路徑

  • σ ≤ 0.5(§3a 典型預算):維持預設 ols_forward——偏差 0.038%(σ=0.2)～ −0.017%(σ=0.5),落在 50-seed 蒙地卡羅雜訊底內、符號不穩,遠低於任一容差門檻(20/15/10%)。
  • σ ≥ 1(低品質 SAD / 稀疏取樣):啟用 wls_inverse(γ 預設 1 即可),~9% CV 改善且 RMSE 全域最佳。
  • 不建議把預設全面換 ols_inverse(RMSE 最差)。

真實 SAD quality 到位後(現成命令)

  1. dcc_cal --seq <序列> --qsigma 取 p̂ / σ̂₀ / r² / bins;檢查 r²(單序列 ~480 樣本 r²≈0.7–0.8 正常;r²<0.5 或 p̂≤0 不上 WLS)。
  2. γ̂ = 2p̂ 換算權重指數(本合成資料 γ̂≈0.8,仍在安全範圍)。
  3. 寫回 config regression.fitter 與 γ;噪聲在典型預算內則維持 forward。

成果 ② · chart 距離公差換算 · 模型

兩個獨立誤差源,經 nl 這個傳導係數耦合

DAC = a + b / dist_cm

兩點標定(鏡頭專屬 INF/MACRO):b = (dac₁−dac₂)/(1/dist₁−1/dist₂),  a = dac₁ − b/dist₁

  • ① 曲線非線性「合焦偏移(DAC)→ DCC 誤差」:已由 --scan + synth nl 驗證。
  • ② chart 物理距離「cm → 合焦偏移(DAC)」:原僅文字工序、程式零實作 —— 本階段補上(chart_dist)。
  • 因果:二者是平行的獨立誤差源,只有透過 nl 耦合才顯現;nl=0 時 DCC 對 chart 距離完全不敏感。
