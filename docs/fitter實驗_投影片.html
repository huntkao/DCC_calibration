理論思路 1/2

問題本質:噪聲全在自變數 —— Errors-in-Variables

  • 回歸模型 DAC = k·disp + b:DAC 是下給 VCM 的指令(無噪), disparity 是 SAD 量測(含噪)——噪聲全在自變數 x。
  • OLS 假設「x 精確、y 含噪」被反轉 → 前向斜率有系統性衰減偏差(attenuation)。
  • 解法直覺:把回歸方向反過來。擬合 disp = a·DAC + c(噪聲回到因變數,OLS 假設成立),再取 k = 1/a。
  • 反向後 quality 可經 q→σ 標定轉成逐樣本權重 → WLS 進一步降變異。
定位:實驗導向。預設 fitter 不動(位元級不變),新增可組態 fitter 以蒙地卡羅量化改善,依數字決定是否換預設。

數學原理 1/3

四種 fitter 的閉式解(x = disp、y = DAC,中心化矩 Sxx、Sxy、Syy)

fitter模型斜率 k偏差性質
ols_forward(v1 預設) 擬合 y = k·x + bSxy / Sxx x 含噪 → 衰減偏差(恆偏小)
ols_inverse 擬合 x = a·y + c,取 k = 1/aSyy / Sxy 消衰減;比值反轉引入小幅正偏(二階)
wls_inverse 反向 + 逐幀權重 wf = qfγSwyy / Swxy 消偏 + 最小變異(權重 ∝ 1/σ² 時)
deming 正交式 EIV,δ = σ²DAC/σ²disp閉式解(含 δ) δ=0 恆等反向(UT 位元級釘死);δ→∞ ≡ 前向
四者共通契約:截距統一 b = ȳ − k·x̄;r² 用 Pearson(對稱); 樣本不足 → E-D03、斜率非正 → E-E01;WLS 之 w ≤ 0 / NaN 樣本形同剔除(= SPEC-004 §3a.1 的 q=0 語意),Σw > 0 否則 E-D03。
工程約束:統一加權累加器,非 WLS 時 w ≡ 1.0(IEEE 恆等乘法)—— 前向路徑浮點運算順序與 v1 逐 bit 相同,既有 80 測試零改動全綠是最強證據。

數學原理 2/3

前向衰減偏差:三步推導 + 假設模組數值驗證

① 噪聲與 y 獨立 → 共變異不受影響

x̃ = x + ε,ε ~ N(0, σ²) ⇒ E[S̃xy] = Sxy

② 但 x 的變異被噪聲灌大(中心化損失 1 自由度)

E[S̃xx] = Sxx + (n−1)·σ²

③ 斜率期望值 → 衰減因子 att

E[k̂fwd] ≈ k · Sx²Sx² + σ²·(n−1)/n = k · att   (Sx² = 真值 disp 母體變異)

0.2−0.02%+0.04%(雜訊底內)
0.5−0.15%−0.02%(雜訊底內)
1.0−0.59%−0.41%
2.0−2.33%−2.30% ✓

假設模組:sweep 10 點、Sx² = 151.2 raw px²(disparity 動態範圍 ±19.3 raw px)。 大 σ 時理論與實測吻合;小 σ 時偏差訊號低於 50-seed 蒙地卡羅雜訊底。

二階效應(測試容差的由來):k̂ 是隨機比值,E[分子/分母] ≠ E[分子]/E[分母] (Jensen 修正)。σ=2、n=10 時 ≈ +0.55% 系統性上修——UT-F2 斷言容差 0.008 即據此 + 跨平台 RNG 緩衝校準,非隨意放寬。

數學原理 3/3

q→σ 標定:冪律回推 + γ = 2p 橋接 + 槓桿修正

σ(q) = σ₀ · q−p  ⟹  w = 1/σ(q)² ∝ q2p  ⟹  γ = 2p̂
  • 程序:反向 OLS 取殘差 → 按 q 等頻分箱(8 箱)→ 每箱 σ̂ = RMS(|e|)→ log σ̂ 對 log q̄ 直線擬合:斜率 = −p、截距 = ln σ₀。
  • Sim 誠實模型 σeff = σ₀/√q ⇒ 真值 p = 0.5 ⇒ γ = 1,兩半題目閉環自洽。
  • 真實 SAD quality(曲線形狀導出,語意未知)到位後:跑同一程序即得該資料的 γ,不需對方提供公式。
⚠ 槓桿修正(實作勘誤):n=10 的擬合殘差被 hat matrix 收縮, e = (I−H)ε——高槓桿且高噪的端點被「吃掉」最多,直接用 |e| 會把冪律壓平 (p̂ ≈ 0.24)。修正:|e| / √(1−hff)(studentized)→ p̂ ≈ 0.40。

DCC 回歸 fitter 精度研究實驗結果
過程洞察

三個計畫外發現(都已修復、勘誤入文件)

① 擬合殘差 ≠ 噪聲

n=10、2 參數擬合的殘差被 hat matrix 收縮 e=(I−H)ε;端點槓桿 h=0.345 又正是高噪(低 q)處 → 冪律被壓平,p̂ 掉到 0.24。

修復:studentized 修正 |e|/√(1−h_ff) → p̂ 0.40。

教訓:小樣本殘差要先還原收縮,才能當噪聲樣本用。

② 排序鍵污染分箱

std::sort 對 pair 是字典序:q 重複時以 |e| 為次鍵 → 等頻箱邊界把同 q 樣本「按殘差大小」切開,製造假斜率,p̂ 一度為 −0.18(負值!)。

修復:stable_sort 僅比較 q。

教訓:排序比較器要最小化——多比一個欄位就是多一個隱性偏差源。

③ 蒙地卡羅斷言的跨平台餘裕

std::normal_distribution 在 libc++(macOS)與 libstdc++(Linux) 產生不同序列——固定 seed 只保單平台確定性;貼線容差(0.25×SE)換平台近半機率翻紅。

修復:以理論二階項 + 1000 種子代理實驗校準 (UT-F2 容差 0.008、UT-Q2 下界 0.28),證據寫進註解。

教訓:MC 斷言容差 = 系統項中心 + ≥3×SE,並以獨立重抽樣視角設定。

結論與建議

決策路徑:依實測噪聲選 fitter;quality 到位後三步標定

量測 / 估計
disparity 噪聲 σ殘差 RMS 或 qsigma 之 σ̂₀
→
σ ≤ 0.5(§3a 預算內) 維持預設 ols_forward——偏差 < 0.04%,低於任一容差門檻 (metal-shield 20% / 2×1 OCL 15% / dual-PD 10%)兩個數量級以上,不必切換。
/
σ ≥ 1(低品質 SAD / 稀疏取樣) 啟用 wls_inverse,γ = 1——不必精確標定即有 ~9% CV 改善、RMSE 全域最優。 勿用純 ols_inverse(RMSE 最差)。

真實 SAD quality 到位後的標定步驟(離線,一次性):

1. calibrate()實際序列殘差(含槓桿修正)→ p̂、σ̂₀、r²; r² 對照基準 0.979,過低表示 quality 與 σ 無穩定冪律關係
→
2. γ̂ = 2p̂系統性偏低(~0.8)無妨——γ 誤標不敏感已由掃描證實,γ=1 亦可
→
3. 寫回 configregression.fitter = "wls_inverse"、weight_gamma = γ̂
本次不換預設的依據:換 fitter 的收益(消偏)只在 σ ≥ 1 顯現且應以 WLS 形式取得; 在預算內噪聲下任何切換都是無收益的行為變更。預設 ols_forward 保持 v1 位元級不變。
